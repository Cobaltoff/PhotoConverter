# 📷 Photo Converter

A desktop application for converting product photos for e-commerce platforms like Trendyol, N11, and Hepsiburada.

## Features
- Bulk photo conversion
- Platform-based profiles (Trendyol, N11, Hepsiburada)
- Custom profile creation and management
- Watermark support (text + logo) with position control
- JPEG/PNG format support with quality control
- Drag & drop support
- Crop, resize, pad and stretch modes
- Batch rename
- EXIF metadata stripping
- Dark/Light theme
- Turkish/English language support
- Preview panel with fullscreen view

## Usage
Download and run the `.exe` file directly. No installation required.

## Tech Stack
- C# + WPF (.NET 8)
- SixLabors.ImageSharp
- Newtonsoft.Json
